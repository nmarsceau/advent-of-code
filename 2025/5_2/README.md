## Testing

```
javac -d . *.java
java com.nmarsceau.advent_of_code_2025_day_5_part_2.Solution < input.test
```

## Solving

Create a file called `input.dat` containing your Advent of Code puzzle input.
Run the following commands to see your solution.

```
javac -d . *.java
java com.nmarsceau.advent_of_code_2025_day_5_part_2.Solution < input.dat
```

## Notes

Tested with OpenJDK 25.0.1 running in Ubuntu.
